--- layout: main title: Navigator category: plugins ---

This plugin provides a simple way to attach the next and previous actions.

Options

name description default
disabledClass Class applied to disable the navigation element disabled
prev Element that will receive the "previous" function
next Element that will receive the "next" function

Callbacks

name description arguments
beforePagination It is called before any pagination, it happens before any animation. To see the format of event go to Creating a plugin for more reference track, event

Installation

{% highlight js %} track.install(new SilverTrack.Plugins.Navigator({ prev: $(".my-track a.prev"), next: $(".my-track a.next") })); {% endhighlight %}
