This plugin provides an integration with ResponsiveHub, a plugin that defines it self as "JavaScript goodies for Responsive Design".

This plugin requires ResponsiveHub js and Moderniz.mediaqueries. You will find the Modernizr feature in the lib directory of SilverTrack repository but you could use your own Modernizr files.

Options

name description
layouts The resolution names configured in responsiveHub. You really should read the responsiveHub readme

Callbacks

name description arguments
onReady The resolution names configured in responsiveHub. You really should read the responsiveHub readme track, options, event
onChange The resolution names configured in responsiveHub. You really should read the responsiveHub readme track, options, event
